Principles

보이는 인상보다 오래 남는 감각을 설계합니다.

01

Simple by default

복잡한 설명보다 한 번에 이해되는 구조를 우선합니다. 첫 화면부터 다음 행동이 보이도록 만드는 것이 기준입니다.

02

Atmosphere with control

분위기는 감성만으로 만들지 않습니다. 컬러, 간격, 타이포의 리듬으로 브랜드의 결을 조정합니다.

03

Built for return visits

한번 보고 잊히는 페이지가 아니라, 다시 들어왔을 때 낯설지 않고 오래 기억되는 흐름을 지향합니다.
